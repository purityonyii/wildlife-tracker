Project Description

This Wildlife Sighting Tracker is a web application built using Express.js. It loads wildlife observation data from a JSON file and it provides an API endpoints to view, filter, and search wildlife sightings.

API Endpoints

  • GET /api/sightings – Get all sightings
  • GET /api/sightings/verified – Get verified sightings
  • GET /api/sightings/species-list – Get unique species list
  • GET /api/sightings/habitat/forest – Get forest sightings
  • GET /api/sightings/search/eagle – Search for eagle sightings
  • GET /api/sightings/find-index/moose – Find index of Moose
  • GET /api/sightings/recent – Get 3 most recent sightings
